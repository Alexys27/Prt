Managementul pacienților unui spital

🏥

Acesta a fost primul proiect mare pe care l-am realizat în anul II de studiu la facultate. Proiectul presupune realizarea de acțiuni asupra pacienților unui spital. A fost realizat utilizând limbajul de programare C și IDE-ul Visual Studio Community. Proiectul este împărțit pe mai multe module, separând și organizând fiecare funcție necesară realizării lui.

Acest proiect este realizat pentru consolă, neavând un UI. Folosește fișiere binare pentru reținerea pacienților și are un meniu pentru a facilita executarea de acțiuni. Acțiunile posibile în acest proiect sunt cele de afișare a pacienților, modificarea, adăugarea și ștergerea acestora. Realizarea proiectului a fost o adevărată provocare pentru mine și totodată un punct de start. Din acest moment mi-am dat seama că asta este ceea ce îmi doresc să fac și am început să învăț mult mai în amănunt pentru a mă putea perfecționa

Software de Salarizare

💰

Acest proiect este din nou unul realizat pentru cursul de Analiza Sistemelor Informatice Economice de la facultate, în anul III. De această dată, tehnologia folosită este Python, utilizând biblioteca Tkinter pentru realizarea interfaței utilizator. Proiectul presupune alegerea unei companii pentru care a trebuit să realizez un software cu obiectivul digitalizării unui departament al companiei respective. Departamentul ales a fost cel de resurse umane, iar eu am considerat că digitazizarea sistemului de salarizare ar fi secțiunea potrivită pentru realizarea acestui proiect.

Software-ul de salarizare permite acțiuni asupra angajaților companiei (adăugare, modificare, ștergere), pontajul, adăugărea de rețineri (impozite și alte rețineri) și realizarea de rapoarte (stat de plată, fluturaș de salariu, concedii etc.). Toate aceste inserări, modificări, ștergeri și interogări se realizează prin executarea de instrucțiuni SQL asupra bazei de date la care este legată aplicația.

Manager Parole

🔑

Managerul de parole a fost realizat în urma parcurgerii unui curs destinat limbajului Python, folosind din nou biblioteca Tkinter pentru realizarea UI-ului.

Aplicația presupune salvarea datelor de conectare pe diferite platforme și, totodată, poate genera o parolă (butonul „Generate Password”) care îndeplinește toate cerințele de securitate ale platformelor, parola generată este salvată automat în clipboard. Toate aceste date sunt salvate într-un fișier de tip JSON și pot fi accesate prin inserarea numelui platformei în prima casetă de text și apăsarea butonului „Search”. Datele noi vor fi salvate în urma apăsării butonului „Add”.

Snake Game

🐍

Proiectul „Snake Game” face parte tot din proiectele realizate în urma parcurgerii cursului pentru Python, menționat anterior. De această dată, interfața este realizată folosind biblioteca Turtle, iar acest proiect este realizat prin OOP fiecare componentă a jocului având o clasă specifică.

Jocul nu este diferit de clasicul Snake Game pe care îl cunoaștem cu toții, obiectivul realizării acestui proiect a fost familiarizarea cu OOP și interfața grafică. Datele cu privire la scor sunt reținute într-un fișier text pentru reținerea celui mai mare scor, iar controalele jocului sunt tastele „WASD”. Mâncarea șarpelui este generată folosind funcția random asupra coordonatelor la care aceasta trebuie să apară.
